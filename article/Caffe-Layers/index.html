Convolutional Architecture for Fast Feature Embedding Layers.

       caffe要实现神经网络的前向以及反向传播计算需要两个要素：一个是数据，一个是算法。
       先说数据:caffe定义了blob类，用来存储训练时的数据。然后是算法：首先我们知道深度学习有很多种类型的层，比如卷积，pooling，RELU等。

Data Layers

Data

参数 描述 备注
source 数据库文件的路径 必填
batch_size 网络单次输入的数据量 必填
rank_skip 跳过开始的rang_skip*rand(0,1)个数据，通常在异步随机梯度下降里使用 可选
backend 选择使用LEVELDB还是LMDB，默认使用LevelDB 可选

MemoryData

参数 描述 备注
channels 通道数 必填
batch_size 网络单次输入的数据量 必填
height 图像高度 必填
width 图像宽度 必填

HDF5Data

参数 描述 备注
source 读取的文件的路径和文件名 必填
batch_size 网络单次输入的数据量 必填

ImageData

参数 描述 备注
source text文件的路径名，此文件每一行储存一张蹄片的路径名和对应的标签 必填
batch_size 每一次处理的数据个数，即图片数 必填
rank_skip 跳过开始的rang_skip*rand(0,1)个数据，通常在异步随机梯度下降里使用 可选
shuffle[default:false] 是否打乱图片顺序，默认为False 可选
new_height 根据设置的值，输入的图片将会被调整成给定高度 可选
new_width 根据设置的值，输入的图片将会被调整成给定宽度 可选

Convolution Layers

参数 描述 备注
num_output 卷积核的个数 必填
kernel_size 卷积核大小 必填
weight_filler 指定参数的初始化方案 必填
lr_mult 学习率系数，最终的学习率 = lr_mult *base_lr,如果存在两个则第二个为偏置项的学习率，偏置项学习率为权值学习率的2倍 可选
decay_mult 权值衰减 可选
bias_term 是否给卷积输出添加偏置项，默认为true 可选
bias_filler 偏置项初始化，设置为“constant”:0 可选
pad 指定在输入图像周围补0的像素个数，默认为0 可选
stride 指定卷积核在输入图像上滑动的步长，默认为1 可选
group 指定分组卷积操作的组数，默认为1组。如果大于1，我们限制卷积的连接操作在一个子集内。如果我们根据图像的通道来分组，那么第i个输出分组只能与第i个输入分组进行连接。 可选
dilation 扩张卷积参数，制定该大小以后则会在进行卷积计算时，添加相应的空洞 可选

Pooling Layers

参数 描述 备注
kernel_size 指定池化的窗口的高度和宽度 必填
pool 池化方法，默认为MAX,还有AVE或STOCHASTIC（按照概率取，数据越大被取到概率越大） 必填
pad 指定在输入图像周围补0的像素个数，默认为0 可选
stride 指定卷积核在输入图像上滑动的步长，默认为1 可选
new_width 根据设置的值，输入的图片将会被调整成给定宽度 可选

InnerProduct Layers

参数 描述 备注
num_output 全连接层输出节点，或滤波器的个数 必填
weight_filler 指定参数的初始化方案，默认类型为constant,值为0 必填
bias_filler 指定过滤器的偏置类型，默认类型为constant,值为0 可选
bias_term 是否给卷积输出添加并训练偏置项，默认为true 可选

ReLU Layers

参数 描述 备注
negative_slope 设置激活函数在负数部分的斜率。输入数据小于零的部分乘以这个因子，斜率为0时，小于零的部分完全滤掉。默认值为0。 可选

Sigmoid Layers

Sigmiod层使用sigmoid(x)计算每个输入数据的输出

SoftmaxWithloss Layers / Softmax Layers层

       softmax-loss层和softmax层计算大致是相同的。
       softmax是一个分类器，计算的是类别的概率（Likelihood）
       假设我们有一个数组，V，Vi表示V中的第i个元素，那么这个元素的Softmax值就是
       
       也就是说，是该元素的指数，与所有元素指数和的比值
       分类可以计算每个分类的概率 分类通道 N(分类数)H（图像高度）W（图像宽度），可以得出分类数的概率
       softmax-loss layer：输出loss值
       在神经网络的计算当中，我们经常需要计算按照神经网络的正向传播计算的分数S1，和按照正确标注计算的分数S2，之间的差距，计算Loss，才能应用反向传播。Loss定义为交叉熵
       
       取log里面的值就是这组数据正确分类的Softmax值，它占的比重越大，这个样本的Loss也就越小

Accuracy Layers

参数 描述 备注
include:phase 输出分类（预测）准确度，只有test阶段才有因此需要加入include参数 必填

BatchNorm Layers

       BN就是在神经网络的训练过程中对每层的输入数据加一个标准化处理。

       在caffe网络中 最后一步γ*s2 +β由scale层实现，因此在caffe中scale层一般跟在BatchNorm层后

reshape Layers

       层类型：Reshape
       在不改变数据的情况下，改变输入的维度

Dropout Layers

参数 描述 备注
dropout_ratio 过拟合丢弃数据的概率，默认值为0.5。 可选

Eltwise Layers

       Eltwise层的操作有三个：product（点乘）， sum（相加减） 和 max（取大值），其中sum是默认操作： 实现element_wise的A+B，即A和B的对应元素相加。

Crop Layers

       主要作用进行裁剪，使用第二个bottom的尺寸裁剪第一个bottom
       A(20,50,512,512) B(20,10,256,256)
       从第二个轴开始进行裁剪
       C = A[:,25:25+B.shape[1], 128+B.shape[2],128+B.shape[3]]

InterP Layers

       对图像进行插值运算

Concat Layers

       把多个bottom拼接为1个output
       对于两个bottom N1C1W1H1和bottom N2C2W2H2
       可以对N和C进行拼接 W和H一定要一样 concat_dim

maxout Layers

       同时连接多个激活函数，选择激活值最大的函数为输出，因此参数会比一个激活函数多k个，因此参数会成倍增加，该层可以看做是对激活层的学习
